Configurando o Branchy

  • Setup Local
  • Integração n8n
  • Configuração Supabase
  • Chaves de API

Setup Local

Siga os passos abaixo para rodar o Branchy em sua máquina local:

# 1. Clone o repositório
git clone https://github.com/TheSp3ctre/branchyio-main
cd branchyio-main

# 2. Instale as dependências
npm install

# 3. Configure o .env
cp .env.example .env

# 4. Inicie o servidor
npm run dev

Integração n8n

O Branchy usa o n8n para orquestrar o pipeline de análise. Você precisará importar os workflows:

  • Repo Analysis: analyze_repo.json
  • Status Polling: get_status.json

Configure a variável VITE_N8N_BASE_URL no seu .env com a URL do seu webhook.

Configuração Supabase

O banco de dados PostgreSQL e a autenticação são gerenciados pelo Supabase. Certifique-se de configurar a tabela Job conforme o schema em prisma/schema.prisma ou execute o SQL manual no dashboard do Supabase.

Chaves de API

Você precisará das seguintes chaves para o funcionamento completo:

  • Anthropic (Claude): Para análise de código.
  • Google (Gemini): Para enriquecimento de dados.
  • GitHub PAT: Para acessar os repositórios.
